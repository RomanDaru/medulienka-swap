## Development

1. Clone the repository:

```bash
git clone https://github.com/yourusername/medulienka-swap.git
cd medulienka-swap
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```
